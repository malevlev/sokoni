# FleetMS

Fleet Management System built with Flask.

## Setup

1. Install dependencies:
```
pip install -r requirements.txt
```

2. Run the app:
```
python app.py
```

3. Open http://127.0.0.1:5000

## Default Login
- Username: `admin`
- Password: `admin123`

> Change the password after first login by updating it in the admin panel or via the shell.

## Features
- Dashboard with fleet overview
- Vehicles – add, edit, delete
- Drivers – add, edit, delete
- Trips – log and manage trips
- Maintenance – track maintenance records
